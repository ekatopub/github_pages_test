#サイト概要

本サイトは、Web 制作における基本的なフロントエンド実装の理解度を示すために制作しました。

HTML / CSS / JavaScript を用いて、レスポンシブ対応のコーポレートサイトを想定しています。

##使用技術

- HTML5
- CSS3（Flexbox / レスポンシブ対応）
- JavaScript（ES6）

##実装で工夫した点

スマートフォン表示では、ハンバーガーメニューを採用。

- メニュー開閉時に is-open クラスで状態管理

- メニュー表示中は body に is-menu-open クラスを付与し、
  背景スクロールを禁止することで操作ミスを防止

- メニュー内リンク、またはそれ以外の場所ををタップした際に自動でメニューを閉じる

お問い合わせフォームでは javascript でバリデーションを行う。

- 前後の余計な空白は削除

- メールアドレスの形式チェック

##アクセシビリティへの配慮

- ハンバーガーメニューの開閉状態をスクリーンリーダーでも正しく認識できるよう、aria-expanded 属性を使用。

- メニューの開閉に合わせて aria-expanded の値を更新

- 視覚的な状態と支援技術向けの状態がずれないように実装
